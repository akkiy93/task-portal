# イシュータイトルの文字化け修正

## 問題
GitHub CLIで日本語のタイトルを設定する際に文字化けが発生しています。

## 修正方法

### GitHub Web UIから修正（推奨）

#### Issue #3のタイトルを修正
1. https://github.com/akkiy93/task-portal/issues/3 を開く
2. タイトルの右側にある「✏️」アイコンをクリック
3. タイトルを「UI改善: ヘッダー部分の機能追加」に変更
4. 「Save」をクリック

#### Issue #2のタイトルを修正（必要に応じて）
1. https://github.com/akkiy93/task-portal/issues/2 を開く
2. タイトルの右側にある「✏️」アイコンをクリック
3. タイトルを「機能追加: 体調記録欄を日報記入欄に変更」に変更
4. 「Save」をクリック

## 正しいタイトル一覧

- Issue #1: バグ修正: タスク追加・編集ボタンの二重送信問題
- Issue #2: 機能追加: 体調記録欄を日報記入欄に変更
- Issue #3: UI改善: ヘッダー部分の機能追加
